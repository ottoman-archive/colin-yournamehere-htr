---

## Historical Context

This masthead marks the **first issue** of the first official Ottoman newspaper. Key facts:

| Aspect | Details |
|--------|---------|
| **Newspaper Name** | Takvîm-i Vekâyi ("Calendar of Events") |
| **Publication Date** | November 1, 1831 |
| **Significance** | First official Ottoman government gazette |
| **Reign** | Sultan Mahmud II (r. 1808-1839) |
| **Period** | Early Tanzimat reforms |
| **Languages** | Initially Ottoman Turkish; later editions included Arabic, Persian, Greek, Armenian, French |
| **Content of Issue 1** | Sultan's imperial tour to Dardanelles and Edirne (May-June 1831) |

The newspaper continued publication until 1922, making it one of the longest-running newspapers in Ottoman/Turkish history.

## Notes on Abbreviation

The abbreviation جا for جمادی الاخر is typical of Ottoman newspaper mastheads where space was limited. Similar abbreviations were used:

| Abbreviation | Full Form | Month |
|--------------|-----------|-------|
| م | محرم | Muharram (1st) |
| ص | صفر | Safar (2nd) |
| ر | ربیع الاول / ربیع الاخر | Rabi' I/II (3rd/4th) |
| جا | جمادی الاول / جمادی الاخر | Jumada I/II (5th/6th) |
| ب | رجب | Rajab (7th) |
| ش | شعبان | Sha'ban (8th) |
| ن | رمضان | Ramadan (9th) |
| ل | شوال | Shawwal (10th) |
| ذق | ذی القعده | Dhu'l-Qa'da (11th) |
| ذح | ذی الحجه | Dhu'l-Hijja (12th) |
